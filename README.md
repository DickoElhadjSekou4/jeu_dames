# 🎮 Jeu de Dames en Python

Un jeu de dames graphique développé en Python avec Pygame.

## ✨ Fonctionnalités
- Plateau graphique 8x8
- Pions avec sélection et mouvements surlignés en vert
- IA qui priorise les captures
- Sons générés avec numpy
- Détection automatique de victoire

## 🕹️ Comment jouer
- **Clic** sur un pion noir pour le sélectionner
- **Clic** sur une case verte pour le déplacer
- L'IA joue les blancs automatiquement

## ⌨️ Raccourcis
| Touche | Action |
|---|---|
| R | Nouvelle partie |
| Echap | Quitter |

## 🚀 Installation

1. Clone le projet :
git clone https://github.com/DickoElhadjSekou4/jeu_dames.git
cd jeu_dames

2. Installe les dépendances :
pip install pygame numpy

3. Lance le jeu :
python main.py
